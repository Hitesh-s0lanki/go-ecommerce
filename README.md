# go-ecommerce

An e-commerce backend written in Go.

> **Status:** early scaffold. The Go module is initialized, but no application code exists yet.

## Requirements

- Go 1.26 or newer

## Getting started

```bash
git clone https://github.com/Hitesh-s0lanki/go-ecommerce.git
cd go-ecommerce
go mod download
```

## Development

```bash
go build ./...   # build
go test ./...    # run tests
go vet ./...     # static checks
gofmt -l .       # list files needing formatting
```
